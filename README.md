#  [zhiaiwan-lodash](https://github.com/qq61538187/zhiwaiwan-lodash)


> 只爱玩lodash基于lodash二次拓展代码库，增加常用工具方法，定义更新同步lodash

```sh 
npm install 
##

npm run build

## 进入 release or release-es 发布到npm

npm publish

## 取消发布 24小时之内的

npm unpublish 包名 --force

## 取消发布指定版不能

pm unpublish 包名@1.12.3
```
